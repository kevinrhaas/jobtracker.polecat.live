How it works

From request to delivered in one place.

Intake a request

Log a new job by hand or import a batch from your spreadsheet or a Microsoft Form. It gets an auto-generated job number and a type-based icon instantly.

Move it through the work

Drag it across the board as it progresses, edit rich details, drop comments, attach files, and request approval — with a full history and one-click undo behind you.

Report & deliver

Watch live KPIs, save the views you check every day, export to CSV/Excel/JSON for the end-of-year report, and share a direct link to any job.

Built to grow

Local-first today, backend-ready tomorrow.

JobTracker starts as a pure browser app so it's fast and private — and it's designed to progressively connect to real infrastructure as the team needs it.

PHASE 1 · NOW

Local-first

Everything in your browser. IndexedDB for attachments, full export/import, six themes, undo & history.

PHASE 2

API bridge

Opt-in connection to a lightweight REST server bridging to SQLite or PostgreSQL.

PHASE 3

Backend-as-a-service

Supabase / PocketBase for auth, role-based access, and managed file storage.

PHASE 4

Edge sync

WebAssembly databases in the browser that sync automatically when online.
